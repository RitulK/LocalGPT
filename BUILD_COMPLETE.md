## 🎯 Router Logic Implemented

The intelligent router analyzes prompts using:

1. **Coding Keywords** (25+ keywords)
   - Detects: code, function, class, debug, python, javascript, etc.
   - Routes to: Coding model (qwen2.5-coder)

2. **Reasoning Keywords** (20+ keywords)
   - Detects: analyze, compare, evaluate, explain, why, how, etc.
   - Routes to: Reasoning model (llama3.2)

3. **Code Block Detection**
   - Detects: ``` in prompt
   - Routes to: Coding model

4. **Length Analysis**
   - Detects: 100+ words
   - Routes to: Reasoning model

5. **Default Fallback**
   - No special pattern → General model
